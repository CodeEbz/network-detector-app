# Network Detector (React)

Simple React app that detects Nigerian mobile network providers from phone number prefixes.

Quick start

1. Install dependencies

```bash
cd network-detector-react
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

Push to GitHub

```bash
git init
git add .
git commit -m "Add network detector React app"
git branch -M main
git remote add origin <your-git-repo-url>
git push -u origin main
```

Hosting

- Use Vercel or Netlify and connect your repository. For Vercel, push to GitHub then import repository. Vercel auto-detects Vite and builds with `npm run build`.
- Or use GitHub Pages by building and publishing the `dist/` folder using an action or a static host.
